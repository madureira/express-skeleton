#express-skeleton

This is an small skeleton application written in [Nodejs!](https://nodejs.org/) on [Express.js!](http://expressjs.com/).


## Dependencies

1. Node >= 0.12.2


## How to install?

```sh
     $ npm install
```


## How to Run?

```sh
    $ npm start
```


## How to access?

[http://localhost:3000!](http://localhost:3000/)


### Running unit tests

```sh
    $ npm test
```
