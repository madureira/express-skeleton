#express-skeleton

[![GitHub version](https://badge.fury.io/gh/madureira%2Fexpress-skeleton.svg)](http://badge.fury.io/gh/madureira%2Fexpress-skeleton) [![Dependency Status](https://www.versioneye.com/user/projects/554939fe5d4f9a0b990012c3/badge.svg?style=flat)](https://www.versioneye.com/user/projects/554939fe5d4f9a0b990012c3)

This is a small skeleton application written in [Nodejs](https://nodejs.org/) on [Express.js](http://expressjs.com/).


## Dependencies

1. Node >= 0.12.2


## How to install?

```sh
     $ npm install
     $ npm install -g karma-cli
```


## How to Run?

```sh
    $ npm start
```


## How to access?

[http://localhost:3000/](http://localhost:3000/)


### Running unit tests

```sh
    $ npm test
```
